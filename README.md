# tableInfo2ExcelForMysql
export mysql table detail to excel file

导出mysql某个库中的表信息到excel文件中

格式如下：
	
表名user

序号   字段名    名称      类型          是否可空    key   备注

1     id         id       bigint(20)    NO        PRI	

2     username   用户名   bigint(20)     YES       MUL	

3     passowrd    密码    varchar(50)    YES
